# 🕯️ Candlestick Pattern Detection System for Indian Stock Market

A production-ready full-stack application that detects candlestick patterns in Indian stocks and provides Buy/Sell recommendations with expected returns analysis.

## 🎯 Features

- **Real-time Pattern Detection**: Detects 12+ classic candlestick patterns
- **Indian Market Focus**: Supports Nifty 100, Nifty 50, Sensex, Bank Nifty, Midcap Nifty
- **Smart Recommendations**: AI-powered Buy/Sell signals with confidence levels
- **Expected Returns**: Forecasts returns for 1, 5, 10, and 20-day periods
- **Interactive Dashboard**: Beautiful React UI with real-time candlestick charts
- **REST API**: FastAPI backend with comprehensive endpoints

## 📊 Supported Patterns

- Hammer (Bullish)
- Hanging Man (Bearish)
- Bullish/Bearish Engulfing
- Doji
- Morning/Evening Star
- Shooting Star
- Piercing Line
- Dark Cloud Cover
- Bullish/Bearish Harami
